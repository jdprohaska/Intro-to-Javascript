Arrays with the Select element and Select object

Using information from a select object/drow down list

This page demonstrates some examples of how to use the information you can get from a drop down list. This could include looking up information in a corresponding (parallel) array, using the selected values in a calculation (prices, sales tax, inventory count) or displaying the information on the page.

These examples will use a drop down that has a default option and one that does not. At this point it does not matter if the drop down is hard coded in HTML or populated with Javascript. The presence of the default option does make a difference when using the selectedIndex to look up corresponding values in a parallel array.

The most common uses for information from the SELECT object

  • Use the index of the selected option for looking values in a parallel array
  • Use the value of the selected option in a calculation
  • Use the text or words of the selected option to lookup a value
