Arrays with the Select element and Select object

The HTML Select Element Example

This page demonstrates using HTML to create a drop down box using the HTML SELECT and OPTION elements. For each example the first method uses HTML. The second method uses Javascript and an array to dynamically create the OPTION element.

A Javascript array is commonly used to dynamically create or populate drop down elements on a web page. This allows the page content to change based upon the available information instead of having to be edited every time the content changes.

Example #2 Drop Down using the VALUE attribute - HTML

The value attribute is used to provide information related to the option. This allows you to describe an option to the user and then related information to the server for processing. In this case the option describes a product but the form will send the product number of the selected option to the server.

HTML drop down using Javascript to populate the drop down with a default "Please Select" option. Note the value of the default option is often set to "" or 0. This is used to validate the drop down field. If the value is "" or 0 the user probably did not make a selection.
